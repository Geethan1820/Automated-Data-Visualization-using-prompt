PROJECT TITLE: GVS DataNova (Intelligent Data Analytics)
================================================================================

1. WHAT IS THIS PROJECT?
--------------------------------------------------------------------------------
DataNova is a smart tool that helps you understand data files (like Excel or CSV) 
without needing to be a math or coding expert. You just upload your file and 
chat with the AI to get answers and charts instantly.

2. WHY WAS IT BUILT? (PROBLEM STATEMENT)
--------------------------------------------------------------------------------
Traditional data tools are too hard for many people. Most users struggle to:
- Clean messy data.
- Create professional charts quickly.
- Understand what the numbers are actually saying.

DataNova solves this by using "Natural Language Processing"—meaning you can 
talk to your data like you're talking to a friend.

3. KEY FEATURES
--------------------------------------------------------------------------------
- 🤖 SMART CHAT: Ask questions like "Show me sales by region" to get an answer.
- 📊 AUTO-VISUALS: Beautiful, interactive charts appear automatically.
- 💾 SAVE & RESTORE: Your work is saved. Log in any time to see your old chats.
- 🔒 AUDIT LOGS: The system keeps track of all questions asked for security.
- 💎 PREMIUM THEME: High-end "Midnight Emerald" visual design.

4. TECHNOLOGY USED
--------------------------------------------------------------------------------
- VISUALS: React (Modern JavaScript framework)
- BRAIN: Python & FastAPI
- AI: Google Gemini (Advanced Language Model)
- STORAGE: PostgreSQL Database

5. HOW TO RUN THE PROJECT (SIMPLE STEPS)
--------------------------------------------------------------------------------
1. Ensure you have Docker, Python, and Node.js installed.
2. Start the database: run "docker compose up -d" in the main folder.
3. Start the server:
   - Go to the 'server' folder.
   - Run "pip install -r requirements.txt".
   - Run "python main.py".
4. Open the app:
   - Go to your browser and type: http://localhost:8000
